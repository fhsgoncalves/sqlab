Rust

Blazing Fast

Written in Rust from the ground up. No garbage collector pauses, no JVM warmup, no JavaScript runtime. Every keystroke is instant.

GPUI

Powered by the same GPU-accelerated UI framework behind the Zed editor. Silky smooth rendering, even on large result sets.

Memory

zql sips memory where other editors gulp it. Run it comfortably alongside your dev server, Docker containers, and everything else.

UX

Radically Simple

No bloat. No 200-option settings menus. zql does one thing — lets you write and run SQL — and it does it beautifully.

Full PostgreSQL support out of the box — connect, explore your schema, and run queries instantly. More databases coming soon.

Open Source
